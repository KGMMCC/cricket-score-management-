# ক্রিকমাস্টার: প্রফেশনাল ক্রিকেট ম্যাচ ম্যানেজার (CricMaster)

একটি আধুনিক এবং শক্তিশালী ক্রিকেট ম্যাচ ম্যানেজমেন্ট অ্যাপ্লিকেশন, যা পাড়ার ক্রিকেট থেকে শুরু করে যেকোনো টুর্নামেন্টের জন্য আদর্শ। এটি প্লেয়ার লটারি, টস, লাইভ স্কোরিং এবং এআই (AI) ভিত্তিক ম্যাচ বিশ্লেষণ প্রদান করে।

## প্রধান ফিচারসমূহ:

- 🏏 **প্লেয়ার ম্যানেজমেন্ট:** খেলোয়াড়দের নাম এবং স্কিল (ভালো/সাধারণ) অনুযায়ী তালিকাভুক্ত করা।
- 🎲 **সুষ্ঠু লটারি:** খেলোয়াড়দের স্কিল অনুযায়ী স্বয়ংক্রিয়ভাবে ভারসাম্যপূর্ণ দল গঠন।
- 🪙 **নির্ভুল টস:** ব্রাউজারের শক্তিশালী র্যান্ডম জেনারেটর ব্যবহার করে ১০০% সঠিক টস।
- 📊 **লাইভ স্কোরিং:** রিয়েল-টাইম রান, উইকেট এবং ওভার ট্র্যাকিং।
- 🤖 **এআই বিশ্লেষণ:** ম্যাচ শেষে Gemini AI এর মাধ্যমে বিস্তারিত পারফরম্যান্স রিপোর্ট।
- 📱 **রেসপন্সিভ ডিজাইন:** মোবাইল এবং ডেস্কটপ উভয় ডিভাইসেই স্মুথ ইউজার ইন্টারফেস।

## প্রযুক্তি (Tech Stack):

- **Frontend:** React + TypeScript
- **Styling:** Tailwind CSS + Framer Motion (Animations)
- **Icons:** Lucide React
- **AI Engine:** Google Gemini AI (via @google/genai)
- **Build Tool:** Vite

## কিভাবে ব্যবহার করবেন?

১. খেলোয়াড়দের নাম এবং তাদের খেলার ধরন (Skill) ইনপুট দিন।
২. 'লটারি শুরু করুন' বাটনে ক্লিক করে স্বয়ংক্রিয়ভাবে দল গঠন করুন।
৩. টসের মাধ্যমে ব্যাটিং বা বোলিং সিদ্ধান্ত নিন।
৪. লাইভ স্কোরিং সেকশনে রান এবং উইকেট আপডেট করুন।
৫. ম্যাচ শেষে 'এআই বিশ্লেষণ' বাটনে ক্লিক করে পারফরম্যান্স রিপোর্ট দেখুন।
